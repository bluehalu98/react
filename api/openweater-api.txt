■ 서비스신청
  ● 상단메뉴 api 에서 제공되는 서비스 목록
  ● subscribe(구독) 
    - Pay as you call 및 Professional collections 목록에서 선택
    - 영문주소입력 (영문주소변환 : https://www.jusoen.com/)
    - 신용카드 입력 (카드정보와 동일하게 입력)
  ● 사용할 api목록의 doc파일 확인
  ● 브라우저 주소창에 제공되는 API입력 후 테스트
